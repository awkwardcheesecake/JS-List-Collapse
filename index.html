What it does

This bit of JavaScript / jQuery allows you to make any piece of HTML an interactive, collapsible content block.

How to Use It

  • Download the "js-list-collapse.js" file and link to it in your project. Keep in mind that you have to have jQuery linked to BEFORE this file.

  • Make sure your HTML is properly structured for this plugin, and add the necessary classes. Examples are given in the demo.html file, and a simple example is below:

<div>
    <h3 class="jslc-ClickHeader">Title to click</h3>
    <p class="jslc-subBlock">Content that will collapse</p>
</div>

Important! All headers and sub blocks must share a parent, and only one header / sub block pair should be present per parent element. Think of the parent like box that holds the collapsible content together.

  • Enjoy! That's all you need to get the script working.

How it Works

  • This script adds some code to the $(document).ready() call which searches the html for the "jslc-clickHeader" class. For each element, a "+" or a "-" is generated and placed in a span before the content. This is called the "expand symbol" and reflects whether clicking the element will show or hide the content. The reason for this auto-generation is so if the user has JavaScript turned off on their browser, all of the content will still be visible, as if this plugin had never been installed at all!

  • A simple jQuery .click() event is added to the "jslc-clickHeader" elements that toggles the visibility of the "jslc-subBlock" and toggles the expand symbol between "+" and "-".

  • The header is tied to the first sub block which shares a common parent, so all header / sub block pairs should have a common parent which holds only one header / sub block pair. Coding it this way makes it possible to have as many collapsible elements on a page as you would like, and be able to add or change them however you like without having to uniquely identify each item.
